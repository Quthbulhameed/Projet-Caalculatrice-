# Calculatrice en C

### Contexte

Ce programme implémente une simple calculatrice en utilisant la langue de programmation C. Il prend en entrée deux nombres et un opérateur pour effectuer des opérations mathématiques simples telles que l'addition, la soustraction, la multiplication, la division, le modulo, le factoriel et le PGCD / PPCM.


#### Utilisation

- Le programme peut être exécuté en ligne de commande en fournissant les arguments nécessaires. Le format de la ligne de commande est le suivant : 
#### ./a.out num1 num2 operateur

- num1 représente le premier nombre
- num2 représente le deuxième nombre (nécessaire uniquement pour les opérations nécessitant deux nombres)
- operateur représente l'opérateur mathématique à utiliser (+, -, x, /, %, !, L pour le PGCD et p pour le PPCM)
